# 팀소개
'개발천국'

저희의 팀명은 개발천국입니다. 

저희가 있는 지금 이세상은 개발을 위한 아이디어들로 가득 차 있기에 다양한곳에서 개발 아이디어를 얻자해서 개발천국으로 짓게 되었습니다.

저희는 팀구성원 개인들의 아이디어와 의견들을 자유롭게 나누고 소통하며, 팀으로써 하나의 목표달성을 위해 열심히 노력합니다.

앞으로도 꾸준히 성장하는 개발천국이 되겠습니다.

# 프로젝트 소개
'YouAndMe Schedule'
'유앤미 스케줄'

바쁜 일상 속 건강을 챙기고 싶으신가요?

스트레스를 풀고, 땀을 쭉 빼며 운동을 하고 싶으신가요?

복잡하게 운동을 선택하고, 스케줄을 잡기보단 간편하게 원하는 시간대, 원하는 강사와 예약을 잡고 운동하고 싶으신가요?  

그렇다면 여기 운동하기를 원하는 사람들과 가르쳐주길 원하는 강사들이 모여 편하게 운동예약을 잡을 수 있는 플랫폼인 ’유앤미 스케줄’이 여러분의 운동 스케줄링 설정에 간편함을 제공해드립니다!

운동을 좋아하는 사람이라면 누구나 간단하게 이용해보세요.

# 프로젝트 기획 배경

- 트레이너가 수많은 회원들의 수업 시간을 편리하게 관리할 수 있도록 하기 위해서 서비스를 기획하였다.
- 개인적으로 연락을 주고 받아야 하는 경우가 많으며 수업이 많은 경우 쉬는 날까지도 연락을 통해 일종의 ‘업무’를 해야한다.
- 이에 대해 좀 더 편리함을 제공하기 위해 서비스를 기획하였다.

# 프로젝트 목표

센터,회원,강사간의 운동 스케줄링 시스템을 제공해주는 서버를 구축합니다.

# 사용 기술스택

- Language : Java11
- Framework : SpringBoot 2.7.16
- Database : Mariadb, Redis
- Docker, AWS
- IDE : IntelliJ
- Build: Gradle

# 프로젝트 기능

- 회원가입
    - 회원가입 시 강사, 트레이너 / 일반 중 선택하여 가입을 진행한다.
    - 강사, 트레이너
        - 자격증을 한 개 이상 등록하거나 자신의 센터를 이용해 인증합니다.
- 로그인
    - id와 패스워드를 이용해 로그인 합니다.
    - 로그인 시 가입되어 있지 않은 id이거나 id와 패스워드가 일치하지 않을 때 에러를 발생한다.
- 예약 기능
    - 트레이너는
        - 수업이 가능한 시간대를 설정해둔다.
    - 회원은
        - 수업을 원하는 시간대를 선택해 수업 신청을 한다.
    - 트레이너는 해당 신청에 대해 수락 / 거절을 할 수 있다.
    - 수락 시 트레이너와 회원의 캘린더에 해당 수업 시간이 자동으로 등록된다.
- 친구 추가 기능
    - 트레이너 - 회원 또는 모든 사용자들 간에 친구 추가를 할 수 있다.
    - id, 닉네임을 통해 검색할 수 있다.
- 알림 전송 기능
    - 트레이너가 수업 신청을 수락 / 거절 시 회원에게 카카오톡으로 알림이 발송 된다.
    - 수업 하루 전날에 수업에 대한 리마인드 알림이 자동으로 전송된다.
        - 예) 9/30 토요일 수업인 회원에게 9/29일 알림이 발송된다.
        알림 내용 : ’내일 18:00 PT수업 일정이 등록되어 있음을 알려드립니다 ! “
- 채팅 기능
    - 친구로 등록되어 있을 경우 1:1 채팅이 가능하다.
- 게시판 기능
    - 관리자 권한
        - 글을 작성할 수 있고 게시글에 대한 삭제 권한이 있다.
        - 관리자의 권한으로 작성한 글은 게시판에서 글 목록 가장 상단에 고정된다.
    - 게시글 작성 시에는 작성자로 닉네임이 보여진다.
    - 카테고리 : 자유공간 / 공유공간 / 추천공간
    - 자유공간
        - 모든 사용자가 자유롭게 글을 작성할 수 있다.
        - 글을 등록할 때 ‘악성글, 비판글 등은 삭제될 수 있습니다’는 메시지를 띄운다.
    - 공유공간
        - 운동 관련 영상 / 운동 꿀팁 / 운동 정보 공유 등
    - 추천공간
        - 선생님 추천 / 센터 추천 등의 글을 작성할 수 있다.
    - 게시글에 대한 댓글을 작성할 수 있다.
    - 검색 기능
        - 글제목 / 글내용 / 닉네임 / 해시태그 중 일치하는 키워드가 있는 게시글이 결과로 보여진다.
